#### Método remove 

Este método permite buscar y remover un elemento dado, si el elemento existe, entonces lo quita de la lista.

``` python
lista_enteros = [3, 2, 2.5,  5]

lista_enteros.remove(2)

print(lista_enteros)
``` 
_Salida:_
**> [3, 2, 2.5,  5]**
