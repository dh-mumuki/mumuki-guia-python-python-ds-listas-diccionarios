Este método permite buscar y remover un elemento dado. 
Veamos un ejemplo:

``` python
lista_enteros = [3, 2, 2.5,  5]

lista_enteros.remove(2)

print(lista_enteros)

ム
> [3, 2.5,  5]
``` 

> :memo: **Dada la siguiente lista, elminá el elemento que no corresponda**<br>
