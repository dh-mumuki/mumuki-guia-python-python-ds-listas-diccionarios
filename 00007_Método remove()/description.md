Este método permite buscar y remover un elemento dado. Para hacerlo tenemos que poner entre paréntesis el elemento y no su posición.  
Veamos un ejemplo:

``` python
lista_enteros = [3, 5, 2.5, 2]
lista_enteros.remove(2)
print(lista_enteros)
> [3, 5, 2.5]
```
